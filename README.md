# byaga.dev

Personal portfolio site for Aga Drapinska-Tonge — data analytics engineer, developer, and film reviewer.

## Features

- Terminal-themed UI with boot sequence and system diagnostics
- Expandable project cards with descriptions
- Auto-updating top 10 film rankings from [bayleyfilmclub.com](https://bayleyfilmclub.com)
- Hidden breakout game easter egg
- Responsive design with accessibility support
- Custom favicon and Open Graph meta tags

## Stack

Single HTML file — no build step, no dependencies, no frameworks. Just HTML, CSS, and vanilla JavaScript.
